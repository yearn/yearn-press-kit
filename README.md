A centralized asset bank for a decentralized brand.  
Take the blue pill and enter the world of yearn.  

### Install and run
1. Clone as template from GitHub
2. Run `yarn`
3. Run `yarn run dev`
4. Open the browser and navigate to `http://localhost:3000`
